## ✨ Features

* 📁 Automatic file organization by type (Images, Videos, Documents, Audio, etc.)
* 👀 Preview files before organizing
* ⚡ One-click cleanup
* 🧠 Smart file categorization
* 🖥 Simple and clean desktop interface
* 🚀 Fast and lightweight (no external dependencies)

---

## 🛠 Tech Stack

* **Python**
* **Tkinter** (GUI)
* **OS & File System Automation**

---

## 🚀 How It Works

1. Select a folder
2. Preview files (optional)
3. Click **"Organize Files"**
4. Files are automatically sorted into categorized folders

---

## ▶️ How to Run

### Option 1 — Run with Python

```bash
git clone https://github.com/bdad2446-prog/devclean-pro.git
cd devclean-pro
python gui.py
```

> ⚠️ Make sure Python is installed on your system.

---

### Option 2 — Run as Desktop App (.exe)

You can convert the app into an executable using PyInstaller:

```bash
pip install pyinstaller
pyinstaller --onefile --windowed gui.py
```

The executable will be available in the `dist/` folder.
